# PollPulse - Polling App

## Tech Stack
React, Node.js, Express.js, MongoDB, JWT

## Features
- Create poll
- Vote on poll
- View results

## API Endpoints

POST /createPoll
Creates a new poll.

GET /polls
Returns all polls.

POST /vote/:id/:index
Vote for an option.

## Setup

1. Clone repository
2. Run npm install
3. Run node server.js
4. Open index.html
